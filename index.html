CodeSync

2025-Jan

  • Next.js
  • TypeScript
  • Stream SDK
  • Convex
  • Clerk
  • Shadcn UI
  • Server Components
  • Video Conferencing

CodeSync is a comprehensive video collaboration platform offering real-time conferencing, screen sharing, and recording capabilities within a sleek, responsive interface.

CodeSync is a modern video collaboration platform built with cutting-edge technologies to provide seamless communication experiences for remote teams. Leveraging Next.js with TypeScript for the frontend and Stream SDK for real-time communication, the platform offers crystal-clear video conferencing with multi-participant support, interactive screen sharing of entire screens or specific application windows, and high-quality recording capabilities. The application incorporates Clerk for secure authentication with multiple sign-in methods and role-based authorization, while Convex powers the database and backend for real-time data synchronization. With a responsive design implemented through Tailwind CSS and Shadcn UI components, CodeSync delivers an optimized experience across all devices. The architecture strategically utilizes Server Components for improved performance, Client Components for interactive elements, and Server Actions for secure operations, representing the next generation of remote collaboration tools with user experience at its core.

codesync.vercel.app


Codex Guys

2024-Apr

  • React.js
  • Three.js
  • Physics Engine (Rapier)
  • State Management
  • Game Development
  • React Three Fiber

A challenging obstacle course game built with React Three Fiber and Three.js.

Codex Guys is an immersive 3D obstacle course game that combines physics-based gameplay with smooth character animations and responsive controls. Developed using React Three Fiber and Three.js, the game features a state machine-based game loop, handling transitions from the main menu to countdowns, gameplay, and game-over states. Players experience realistic movement mechanics, including jumping, rotation, and collision detection, powered by the Rapier physics engine. A dynamic camera system ensures an engaging and fluid perspective, keeping the player focused on the course ahead. Whether for casual fun or competitive time trials, Codex Guys delivers an exciting and interactive experience.

codexguys.vercel.app



Skibidi

2024-Jul

  • React.js
  • Three.js
  • React Three Fiber
  • Wiggle Bones
  • Mixamo

A dynamic 3D animation project implementing the Wiggle Bones library with React Three Fiber and Three.js.

Wiggle Bones with React Three Fiber brings physics-driven secondary animations to 3D scenes using React Three Fiber and Three.js. By leveraging the Wiggle Bones library, this project enables natural, springy motions for character models and dynamic objects, adding realism and fluidity to animations. Designed for seamless integration with Mixamo-animated characters, it allows fine-tuning of parameters such as stiffness, damping, mass, and gravity effects for precise motion control. Optimized for performance, it ensures smooth framerates while maintaining high-quality visuals within the R3F ecosystem. Whether for game development or interactive web experiences, this project enhances 3D animations with lifelike responsiveness.

skibidi.vercel.app
